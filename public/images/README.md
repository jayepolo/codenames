# Team Images

Place your team character images in this directory to customize the appearance.

## Image Files

- `red-team.png` or `red-team.jpg` - Image for Red Team
- `blue-team.png` or `blue-team.jpg` - Image for Blue Team

## Recommended Specifications

- **Size**: 256x256 pixels (will be displayed at 64x64)
- **Format**: PNG (for transparency) or JPG
- **Style**: Character portraits, spy icons, or themed graphics
- **Aspect Ratio**: Square (1:1)

## How to Add

1. Save your images with the names above in this directory
2. The app will automatically use them if they exist
3. If no images are found, placeholder icons will be displayed

## Examples

- Spy character portraits
- Team mascot images
- Themed icons (detective, agent, etc.)
- Abstract team symbols
